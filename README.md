## 🍀 주요 기술


- **Backend**
    - Java 17
    - Spring Boot 3.2.0
    - Spring Data JPA **3.2.2**
    - Spring Security **6.2.0**
    - QueryDSL 5.0.0
    - Server-Sent Event
    - WebSocket
- Frontend
    - HTML 5.0
    - CSS
    - JavaScript
- **DB**
    - MySQL 8.0
    - Redis 7.2.4
    - MongoDB 6.0.13
- **Infra**
    - AWS
        - EC2
        - S3
        - RDS
        - Route53
    - Nginx v1.18.0 (Ubuntu)
    - MongoDB Atlas
    - Docker v25.0.0
- **Docs**
    - Swagger
